# 🤖 DevSupport AI

DevSupport AI is an AI-powered software development chatbot built using Python, Streamlit, and the Gemini API. It provides intelligent assistance for programming concepts, debugging, algorithms, web development, databases, AI/ML, and other software engineering topics in simple and beginner-friendly language.

## 🌐 Live Demo

🔗 **App:** https://chatbot-7mexrthmzb4nkympssdsqp.streamlit.app/

## 🚀 Features

*  Interactive AI chatbot for software development
*  Supports Python, Java, C/C++, JavaScript, React, SQL, and more
*  Debug code and explain errors
*  Explain programming concepts and algorithms
*  AI-powered responses using Google's Gemini API
*  Modern and responsive Streamlit user interface
*  Secure API key management using environment variables and Streamlit Secrets

## 🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* HTML/CSS
* python-dotenv

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/samruddhi2402/chatbot.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Create a `.env` file:

```env
GEMINI_API_KEY=your_gemini_api_key
```

4. Run the application:

```bash
streamlit run app.py
```
